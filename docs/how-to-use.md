# How to Use LandIt!

## 1. Sign up and sign in

1. Open the app (e.g. http://localhost:3000).
2. Click **Get started** or **Sign in**.
3. In the modal, choose **Sign up** or **Sign in**.
4. Enter your email and password. For sign up, confirm your email if the project has email confirmation enabled.
5. After signing in, you are taken to the Dashboard.

## 2. Dashboard

1. On the Dashboard you see: total applications, in progress (Wishlist + Applied + Interview), offers, and closed (Rejected + Ghosted).
2. Set your **weekly application goal** (e.g. 5) and track how many you’ve applied to this week.
3. Use **Quick links** to add job board URLs (e.g. Indeed, LinkedIn) for one-click access.
4. Check **Upcoming interviews** for date, company, role, and meeting link.
5. Use the **Pipeline** chart to see how many applications are in each stage.

## 3. Job applications (Kanban)

1. Go to **Job applications** in the sidebar.
2. To add an application: click **Add application**, fill company, role, job link, description, and status. Save.
3. To move an application: drag its card to another column (Wishlist, Applied, Interview, Offer, Rejected, Ghosted).
4. To add interview details: set status to **Interview**, then open the card and set date, type (Phone/Video/Onsite), and meeting link.
5. To attach a resume or cover letter: open the card and choose a file from your resume/cover letter libraries.
6. To hide an application from the board: use the card menu to archive it.

## 4. Job requirements

### Checklist

1. Go to **Job requirements** → **Checklist**.
2. Add items (name, description, link, status: Not started / In progress / Done). Reorder by dragging.
3. Use the progress summary to see how many items are done.

### Resume

1. Go to **Job requirements** → **Resume**.
2. Fill in the resume builder (contact, summary, experience, education, skills, etc.). Use **Preview** to see the result.
3. Use **Export PDF** to download. Use **Save to my resumes** (if available) to store the PDF in your resume library.
4. In **Resume files**, upload PDFs or import from the builder. You can attach one of these to a job application.

### Cover letter

1. Go to **Job requirements** → **Cover letter**.
2. Fill in the cover letter builder. Use **Preview** and **Save to my cover letters** (if available) to store a PDF.
3. In **Cover letter files**, upload or manage PDFs. Attach one to a job application from the application card.

## 5. Interview questions

1. Go to **Interview questions** in the sidebar.
2. Browse categories and questions. Add or edit categories and questions as needed.
3. Click a question to add **Notes** (your answer or talking points). Notes are stored in your browser for that user.
4. Use search and filters to find questions by category or text.

## 6. Settings

1. Go to **Settings** in the sidebar.
2. **Profile:** Update display name and profile photo (upload to avatars bucket).
3. **Password:** Change password (current + new + confirm).
4. **Weekly goal & quick links:** Same settings as on the dashboard.
5. **Demo data:** If you use the demo account, you can add sample applications and checklist items, or remove all demo entries.
6. **Delete account:** Permanently delete your account and all data (confirm by typing DELETE).

## 7. Offer accepted

When you accept an offer, you can use the **Offer accepted** page (linked from the app) for a short congratulations view and a link back to the dashboard.
