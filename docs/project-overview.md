# Project Overview

**LandIt!** is a job application tracker that helps you manage applications, track your pipeline, build resumes and cover letters, and prepare for interviews.

- **Product name:** LandIt!
- **Tagline:** One place to land your next job.
- **Target user:** Job seekers who want a single place to track applications, store documents, and prepare interview answers.
- **Tech stack:** Next.js 16 (App Router), React 19, Supabase (Auth, Postgres, Storage), Tailwind CSS 4.
- **Deployment:** Runs as a Next.js app; requires a Supabase project for backend.
