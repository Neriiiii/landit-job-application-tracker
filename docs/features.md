# Features

- **Landing & auth** — Public landing page (hero, features, how-it-works). Email/password sign up and sign in via modal. Sign out from app sidebar.
- **Dashboard** — Stats (total, active, offers, closed), weekly application goal, job search health, pipeline bar chart, quick links (job boards), aside (upcoming interviews, tip, recent applications).
- **Job applications (Kanban)** — Columns: Wishlist, Applied, Interview, Offer, Rejected, Ghosted. Drag-and-drop, create/edit/delete, interview details, resume/cover letter file links, archive.
- **Job requirements** — Checklist (CRUD, sort, progress). Resume builder (preview, PDF export, file library). Cover letter builder and files.
- **Interview questions** — Categories and questions (CRUD), per-question notes, search/filter, tips.
- **Settings** — Profile photo, display name, change password, weekly goal, quick links, delete account, demo data.
- **Offer accepted** — Congratulatory page. Demo data: seed sample applications and checklist items; remove all demo entries.
