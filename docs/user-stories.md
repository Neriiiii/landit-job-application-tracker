# User Stories

## Landing & authentication

- **As a** visitor **I want to** see a landing page with hero, features, and how LandIt! works **so that** I can decide whether to sign up.
- **As a** visitor **I want to** sign up with email and password **so that** I can start tracking my applications.
- **As a** user **I want to** sign in and sign out **so that** I can access my data securely.

## Dashboard

- **As a** user **I want to** see application stats and a pipeline chart **so that** I understand my job search at a glance.
- **As a** user **I want to** set a weekly application goal and see progress **so that** I stay consistent.
- **As a** user **I want to** see upcoming interviews and quick links to job boards **so that** I can act quickly.

## Job applications

- **As a** user **I want to** add and move applications on a Kanban board **so that** I can track each stage.
- **As a** user **I want to** add interview date, type, and meeting link **so that** I can prepare and join calls.
- **As a** user **I want to** attach a resume and cover letter file to an application **so that** I remember what I sent.

## Job requirements

- **As a** user **I want to** maintain a checklist and see progress **so that** I stay organized.
- **As a** user **I want to** build and export my resume as PDF **so that** I can send it to employers.
- **As a** user **I want to** build and store cover letters **so that** I can reuse and attach them to applications.

## Interview questions

- **As a** user **I want to** browse categories and questions and add my own notes **so that** I can prepare answers.

## Settings

- **As a** user **I want to** set my display name and profile photo **so that** the app feels personal.
- **As a** user **I want to** change my password and delete my account **so that** I can manage my security and data.
